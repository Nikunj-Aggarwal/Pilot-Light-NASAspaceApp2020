Pilot Light Predicting the hotspots of COVID-19 infection by analysing light emissions in population centres using night-time images

Night-time images provide valuable chances to analyse human mobility from on a large scale. In order to make better use of this data and technology, we came up with Project Pilot Light. Pilot Light is a website which examines the recent cases of COVID-19 with information of recent Human activity (received from the NPP VIIRS DNB sensor ). This way we can predict the rise and fall of COVID cases in that particular region. We can integrate real time data to make the website up to date and include factors like CO emission, temperature and population density.
